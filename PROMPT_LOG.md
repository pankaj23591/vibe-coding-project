## Tips for judges (Prompt Quality — 20 pts)

- Include **constraints** (stack, file paths, “no new deps”, etc.).
- Show **iteration**: what failed, what you asked next, what you kept.
- Reference **context**: paste error messages, types, or spec bullets the model should respect.
